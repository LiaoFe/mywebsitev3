## 🛠 Wanna run on your machine?

1. Clone the project

   ```sh
   git clone https://github.com/ironsoul0/ironsoul.lol.git
   ```

1. Install the dependencies

   ```sh
   cd ironsoul.lol
   yarn
   ```

1. Start the development server

   ```sh
   yarn dev
   ```
